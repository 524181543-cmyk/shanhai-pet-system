# 山海经班级宠物积分系统

一个基于山海经主题的班级宠物积分管理系统，学生可以领养专属神兽，通过积分解锁不同的进化形态和技能。

## 功能特性

- **山海经主题宠物**：包含麒麟、凤凰、九尾狐、白泽、应龙等10种山海经神兽
- **宠物进化系统**：每种宠物有4个进化阶段，通过积分解锁
- **技能解锁**：解锁新阶段可获得特效、被动、主动技能
- **积分系统**：支持多种积分获取方式（作业、表现、奖励等）
- **排行榜**：实时积分排行榜，激励学生竞争
- **可视化展示**：精美的宠物形象展示和进化动画

## 技术栈

- **后端**：Flask + Supabase（PostgreSQL）
- **前端**：原生HTML + CSS + JavaScript
- **数据库**：Supabase PostgreSQL
- **图片生成**：AI图片生成（可选）

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 初始化数据库

```bash
cd src/scripts
python init_db.py
```

### 3. 启动服务器

```bash
cd src/scripts
python start_server.py
```

服务器将在 `http://localhost:5000` 启动。

### 4. 访问系统

打开浏览器访问 `http://localhost:5000` 或直接打开 `src/frontend/index.html`

## 系统架构

### 数据库表结构

1. **users** - 用户表
   - 存储用户基本信息和总积分

2. **pet_types** - 宠物类型表
   - 定义山海经神兽的基本信息

3. **pet_stages** - 宠物进化阶段表
   - 定义每种宠物的4个进化阶段

4. **user_pets** - 用户宠物表
   - 记录用户领养的宠物及其状态

5. **pet_skills** - 宠物技能表
   - 定义宠物可解锁的技能

6. **point_records** - 积分记录表
   - 记录所有积分变动

### API接口

#### 用户相关
- `POST /api/users` - 创建用户
- `GET /api/users/<user_id>` - 获取用户信息

#### 宠物相关
- `GET /api/pets/types` - 获取所有宠物类型
- `GET /api/pets/types/<pet_type_id>` - 获取宠物类型详情
- `POST /api/users/<user_id>/pets` - 用户领养宠物
- `GET /api/users/<user_id>/pets` - 获取用户宠物列表
- `GET /api/users/<user_id>/pets/<pet_id>` - 获取宠物详情

#### 积分相关
- `POST /api/users/<user_id>/points` - 添加积分
- `GET /api/users/<user_id>/points/records` - 获取积分记录

#### 排行榜
- `GET /api/leaderboard` - 获取积分排行榜

## 宠物介绍

### 传说级
- **麒麟**：中国传统瑞兽，性情温和，祥瑞之兆
- **凤凰**：百鸟之王，浴火重生，象征永生
- **鲲鹏**：庄子逍遥游中的神兽，鲲之大不知其几千里

### 史诗级
- **九尾狐**：智慧超群，善于幻术
- **白泽**：知万物之精怪，智慧如海
- **应龙**：生双翅的龙，天神之一

### 稀有级
- **毕方**：火灾之兆，火之化身
- **貔貅**：招财聚宝，财富象征
- **饕餮**：贪食凶兽，吞天噬地
- **英招**：天帝花园守护神

## 进化系统

每种宠物有4个进化阶段：

1. **幼崽阶段**（0积分）
   - 初始形态，可爱萌动

2. **成长阶段**（60-100积分）
   - 开始展现特性，获得初步能力

3. **成熟阶段**（180-300积分）
   - 完整体现神兽特征，掌握强大技能

4. **神级阶段**（400-600积分）
   - 最终形态，拥有通天彻地之能

## 技能系统

- **特效技能**：祥瑞光环、神光护体、灵气缭绕等
- **被动技能**：福运加身、智慧启迪、力量增幅等
- **主动技能**：神兽咆哮、元素冲击、幻影分身等
